# Refs Sync

Walk through @docs/citations.quinn.md

Ensure that everything informally linked to has an entry in @paper/refs.bib. Use the checkboxes to help you keep track of what has and hasn't been done. 

Favor arxiv over gatekept journal wherever possible. Do the standard practices for citing blogs and git repos. If a README has a `.bib` entry use that. If a platform like `arxiv` has an exporter/generator for `.bib` entries try to access that. 

Use websearch liberally and DO NOT HALLUCINATE.
